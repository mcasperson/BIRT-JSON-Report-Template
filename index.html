REPORT LINK REPORT DESCRIPTION
Topics per CSP Accepts a list of CSP IDs and a category ID, and returns a stacked bar chart that breaks down the topics assigned to a CSP by the tags assigned to the specified category.
Bugzilla Bugs per CSP Lists the number of bugs and a summary of their status against multiple CSPs.
Table of Topics and Bugzilla Bugs Displays a table of the topics in the supplied CSPs and any Bugzilla bugs associated with them.
Topics per CSP Grouped by Tag Lists the number of topics assigned to a list of CSPs, grouped by the tags in a specified category.
RHEV Topics Included in More Than One Content Specification Lists RHEV topics that are included in multiple CSPs.
